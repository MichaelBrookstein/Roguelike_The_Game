Controls:
W = Move Up
S = Move Down
D = Move Right
A = Move Left

Q = Move Up/Left
E = Move Up/Right
C = Move Down/Right
Z = Move Down/Left

G = Pickup Item
I = Inventory
P = Abilities
Tab = Character Stats
Enter = Move Stairs
Esc = Quit

Mouse over items/characters to see info

Run engine.py to play
